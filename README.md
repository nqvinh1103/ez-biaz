# EzBias (Monorepo)

This repository contains:
- `frontend/` — Vite + React UI (currently using mock API/data)
- `backend/` — .NET backend (to be built)

## Frontend

```bash
npm install
npm run dev
```

Or:

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at: http://localhost:5173

## Backend

Backend will live under `backend/` (planned: .NET 8 Web API + PostgreSQL + JWT access/refresh).
